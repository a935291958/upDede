# 织梦漏洞修复工具

运行环境：python2.6

使用模块：os, sys,shutil, time,chardet

使用说明： 更改config.py里面的配置，根据实际修改：wwwroot、notUp、sonDir、adminList、bakDir、isDelIns参数，可自行添加匹配规则。然后运行**dede.py**
